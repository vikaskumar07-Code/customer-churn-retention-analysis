# customer-churn-retention-analysis
Customer churn and retention analysis using SQL and Python
##  Project Overview
This project analyzes customer churn behavior in a subscription-based business to identify key churn drivers and provide actionable retention insights.

##  Business Problem
The company is facing customer churn. The objective of this analysis is to:
- Identify customers who are churning
- Understand the reasons behind churn
- Provide recommendations to improve customer retention

##  Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib)

##  Dataset
IBM Telco Customer Churn Dataset

##  Key Analysis
- Data loading and cleaning using SQL
- Overall churn rate calculation
- Churn analysis by tenure, contract type, and monthly charges
- Cohort analysis to evaluate customer retention trends

##  Key Insights
- Highest churn observed among customers in the first 3 months
- Month-to-month contracts show significantly higher churn
- Customers with higher monthly charges are more likely to churn

##  Recommendations
- Improve onboarding experience for new customers
- Offer incentives for long-term contracts
- Introduce targeted retention offers for high-risk customers

##  Outcome
This analysis helps businesses understand churn behavior and take data-driven actions to reduce customer attrition.
